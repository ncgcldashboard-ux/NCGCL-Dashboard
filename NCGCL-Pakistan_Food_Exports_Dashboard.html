Section F

Scope, method and interpretation

Source
The dashboard uses the D-10 export table by commodity and country. Source values are in Pak rupees in thousands; US$ values are computed by the dashboard using the FX field in the filter console.
Food scope
The core universe contains — food HS codes across food-relevant chapters. Live animals and other non-food / input-adjacent lines are kept outside the strict core definition.
Optional adjacent
— optional adjacent HS lines were separated for review rather than included automatically. Use this as a clean food-export baseline, not a broader agri-input baseline.
